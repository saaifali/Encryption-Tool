Encryption Decryption tool v1.1
-------------------------------

Authors : Saif Ali and Ankit Singh Bhamra


For Command line version : 
	run tool.py
Can Encrypt or Decrypt single line of text. Results are exported and fetched from files. 
Basic Version of application. Not Recommended for use except for educational purposes.



For GUI version : 
	run toolgui.py
Functionalities : 
1. Encrypt/Decrypt multiples lines of text.
2. Encrypt/Decrypt a single file.
3. Encrypt/Decrypt all files in a folder.

Keep on improving it as much as one can.
